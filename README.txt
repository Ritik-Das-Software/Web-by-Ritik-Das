
SESa – Software Gyancharcha System (Static Website)
===================================================

फोल्डर में फाइलें:
- index.html — मुख्य पेज
- styles.css — डिजाइन/स्टाइल
- script.js — सर्च, स्मूथ स्क्रॉल, इंस्टा लिंक सेटिंग
- assets/instagram.svg — इंस्टाग्राम आइकन

Instagram लिंक कैसे सेट करें?
-----------------------------
1) `script.js` खोलिए
2) यह लाइन ढूँढिए:
   const INSTAGRAM_LINK = "https://instagram.com/ritik_.kushwah";
3) अपने Instagram का पूरा URL डाल दीजिए, जैसे:
   const INSTAGRAM_LINK = "https://instagram.com/ritik";   // उदाहरण

कंटेंट कहाँ डालें?
------------------
- हर सेक्शन `index.html` में एक-एक `<section>` ब्लॉक है (id जैसे #kabir, #geeta).
- उसी ब्लॉक के अंदर आप अपना टेक्स्ट, टेबल, इमेज, वीडियो लिंक जोड़ सकते हैं.
- नई एंट्री/सेक्शन बनाना हो तो `<section class="section" id="...">` कॉपी-पेस्ट करके नए शीर्षक व कंटेंट डालें.
- साइडबार नेविगेशन में लिंक जोड़ने के लिए `<nav class="nav">` के अंदर `<a href="#new-id">नया शीर्षक</a>` जोड़ें.

स्थानीय रूप से कैसे चलाएँ?
-------------------------
- बस `index.html` पर डबल-क्लिक करें (किसी भी ब्राउज़र में खुल जाएगा). कोई सर्वर जरूरी नहीं.

रंग/फॉन्ट कैसे बदलें?
---------------------
- `styles.css` में आप बैकग्राउंड, फॉन्ट, बॉर्डर आदि बदल सकते हैं.
